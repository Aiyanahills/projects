# Мой первый сайт

Я сделал сайт с помощью ASP.Net Core

Есть форма обратной связи, сообщения из которой сохраняются в бд

Для сохранения обратной связи необходимо создать базу данных
В файле appsetings.json нужно заполнить информацию о вашей созданной бд
"Host=localhost;Port=5432;Database=имя_бд;Username=Имя_пользователя;Password=пароль"
